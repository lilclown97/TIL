## 초기화

```
npm init
```

## express install

```
npm i express
```

## mongoose install

```
npm i mongoose
```

## nodemon install

```
npm i nodemon -g
```

## JWT 토큰

```
npm i jsonwebtoken -S
```

## Joi 회원가입 조건

```
npm i joi -S
```

## Sequelize 설정

```javascript
npm i sequelize mysql2 -S
npm i sequelize-cli -D

npx sequelize init

//데이터베이스 생성
npx sequelize db:create
npx sequelize db:migrate
```
