## list

```python
a_list = []
a_list.append(1)     # 리스트에 값을 넣는다
a_list.append([2,3]) # 리스트에 [2,3]이라는 리스트를 다시 넣는다
```

## dict

```python

```

## if / 조건문

```python
if num % 2 == 0: # num을 2로 나눈 나머지가 0이면
        return True   # True (참)을 반환한다.
else:            # 아니면,
        return False  # False (거짓)을 반환한다.
```

## for / 반복문

```python
fruits = ['사과','배','감','귤']

for fruit in fruits:
	print(fruit)

# 사과, 배, 감, 귤 하나씩 꺼내어 찍힙니다.
```
