## list 

- list

```python
let listname = [0, 1, 'name', 3] #숫자, 단일문자는 그냥 선언, 문자열 '' 사용.
```

- 출력

```python
listname[0] #앞에서부터 0번째.
```

- 추가

```python
listname.push('Hi') #listname 마지막에 'Hi' 추가.
```
