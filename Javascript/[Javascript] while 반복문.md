## while 반복문

---

> **while**

- `for`문과 같은 반복문으로 전역 변수를 조건 비교에 사용하고 그 후에도 전역 변수를 변경하여 사용할 때 유용하게 사용할 수 있다.

```javascript
while (조건) {
  실행;
}
```

<br>

> **예제**

- `while`문 사용 예제

```javascript
let num = 1;

while (num <= 5) {
  console.log(num); // 1, 2, 3, 4, 5
  num++;
}

console.log(num); // 6
```
